## Summary of the new issue creation automation

This document provides a summary of the automated issue creation process for low hanging fruit bugs.

- Find a feature/bug to work on from this repository
- Create a GitHub issue on the repository to start working on it

This issue is limited only to the creator of this issue. This means that only the issue author can attempt to solve this issue. If you would like to work on it, please create another issue with the same contents and refer to issue #743 for more information.

## Automation details
- The automation should scan the repository for potential bugs
- It should create new issues based on the findings
- Each new issue should contain the limitation clause
