# Site Survey for a Quiet Merge

First, scope is set like string across a room,
four brass pins holding the corners true.
No promise leaves the table as perfume;
each line is named, each load is passed in view.

The ticket opens. Dust lifts off the plan.
A failing path is measured, not accused.
The proof is small enough to hold in hand:
one trace, one test, one thing that can be used.

At review, light moves slowly over grain.
A note is cut where roughness breaks the run.
The patch returns with cleaner joints again,
and every hidden edge is brought to sun.

Then merge arrives without a victory hymn,
only the click of records fitting square.
The branch goes quiet, runners settle in,
the room remembers why the fix is there.

When payout comes, it is not theatre,
but settlement for care made visible:
FreelanceFlow keeps scope and evidence clear,
and trust is rebuilt from work reviewable.
