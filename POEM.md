# Proof Harbor

The brief arrives as coordinates,
a narrow light on a quiet board;
scope is the rope we tie first,
before any engine starts.

Tests come in like measured tide,
green marks rising line by line;
each failing edge gets named,
then folded back into the chart.

Review is a patient signal,
not applause, not noise;
it asks for proof that holds
when the screen is no longer warm.

The merge gate opens only once
the receipt matches the route:
issue, patch, evidence, note,
all carried in the same clean hand.

Then payout is not a promise shouted,
but a ledger entry made plain;
work leaves the harbor accounted for,
and trust has something it can count.
