# The Quiet Patch

In the glow where terminals breathe,
a small green cursor keeps its vow;
it trims the noise, it names the fault,
and turns the failing test around.

A branch begins as borrowed doubt,
then learns the shape of something true;
each careful diff, each sharper line,
lets hidden trust come into view.

No trumpet sounds for simple code,
no banner drops when bugs depart;
but safer paths are built like this—
one honest patch, one patient heart.
