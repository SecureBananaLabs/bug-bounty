# Merge Window

Brief opens at dawn,
a small square of work made visible:
scope, budget, deadline, light.

Proposal answers in clean lines,
terms lined up like passing tests,
the ask becoming a path.

Message keeps the window warm,
questions pinned, files attached,
status turning from maybe to now.

Test walks the edge of the branch;
review reads what changed,
and trust learns the diff.

Payout lands without theater,
one state closed, the next one green,
FreelanceFlow humming in merge light.
