# The Canary Ledger

In a repo lit by midnight tests,
a canary taps the glass;
it sings where brittle edges rest,
where quiet warnings pass.

Each issue is a lantern raised,
each patch a careful thread;
we map the paths the shadows praised
and make the dark less fed.

So let the build grow clean and bright,
let trust be earned, not claimed;
a small yellow song through the night:
secure by being named.
