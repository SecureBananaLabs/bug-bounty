# OSS Code

In terminals dark, a coder's light,
Where logic bends through sleepless night.
Three stanzas build, then four take flight—
A PR that dares to make things right.

The linter sighs, the tests all pass,
A rebase dance on fragile grass.
Reviewers nod through coffee glass,
"Merged!" they type—the moment's class.

Open source, a garden grown
Not by one, but seeds we've sown.
Each commit stands, though not alone—
A boulder built from every stone.
