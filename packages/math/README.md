# Math Package

Mathematical utilities for the FreelanceFlow monorepo.

## PI Calculation

This package includes tools for calculating and working with the mathematical constant π (pi).

### Key Facts About π

- π is an irrational number — it cannot be expressed as a ratio of two integers
- π is transcendental — it is not a root of any non-zero polynomial with rational coefficients
- The exact value of π has infinitely many decimal places without repetition
- Current world record: 100 trillion digits calculated (as of 2022)
