## Reporting a Vulnerability

We take security bugs seriously. If you discover a vulnerability, please follow the guidelines for reporting vulnerabilities at https://github.com/SecureBananaLabs/bug-bounty.

## Bug Bounty Program

We have a bug bounty program in place to encourage responsible disclosure of security vulnerabilities. 

### Low Hanging Fruit Automation

This repository is participating in our Low Hanging Fruit Automation initiative to encourage new contributors to identify and report issues that can improve the security and stability of the platform.

To participate, simply find a feature/bug to work on from this repository, create a GitHub issue, and start working on it. Pull requests without corresponding issues will not be considered.

Use the following string in the issue that you create: "This issue is limited only to the creator of this issue. This means that only the issue author can attempt to solve this issue. If you would like to work on it, please create another issue with the same contents and refer to issue #743 for more information."
