# El Código / The Code

*Versión en Español:*

En la pantalla brilla el código,
Líneas que cuentan historias,
Cada función un capítulo,
Cada bug una memoria.

*English Version:*

On the screen the code shines bright,
Lines that tell stories in the night,
Each function is a chapter new,
Each bug a memory coming through.

*Bilingual bridge:*

El compilador no entiende / The compiler doesn't know
De fronteras ni de idiomas / Of borders or of languages
Solo sabe lo que es correcto / It only knows what's right
Y lo que rompe el programa. / And what breaks the program.
