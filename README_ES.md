# Lee esto primero

Este repositorio es parte de un sistema de recompensas por contribuciones. Si encontraste un bug, creaste una solución o simplemente quieres contribuir, ¡bienvenido!

## Cómo contribuir
1. Encuentra un bug o feature en el código
2. Crea un issue describiendo el problema
3. Espera la confirmación del equipo
4. Crea un Pull Request con tu solución
5. ¡Recibe tu recompensa!

Gracias por contribuir al código abierto.
