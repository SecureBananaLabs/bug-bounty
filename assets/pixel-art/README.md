# Cosmic Cat Pixel Art

**What:** A cosmic cat floating in a purple nebula with twinkling stars and a crescent moon, wearing tiny space goggles.

**Why:** I chose this theme because cats are universally loved in tech culture, and combining that with the mystery and wonder of space creates a whimsical, imaginative piece. The cosmic cat represents curiosity and exploration—qualities essential to both developers and the universe itself. The limited pixel palette forces creative choices that make each star and color gradient meaningful.

**Specs:**
- Canvas: 64x64 pixels
- Format: PNG with transparency
- Colors: Limited 16-color palette
- Style: Retro pixel art with dithering for nebula effects
