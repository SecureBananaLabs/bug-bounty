# Pixel Art: Cyberpunk City

**What:** A 64x64 pixel art depiction of a futuristic cyberpunk cityscape at night.

**Why:** I chose this theme because cyberpunk aesthetics naturally lend themselves to pixel art—the high contrast between neon lights and dark shadows creates visual depth even at low resolutions. The cityscape represents the intersection of technology and creativity, which aligns with the innovative spirit of this project.

## Details
- **Canvas size:** 64x64 pixels
- **Color palette:** Neon cyan, magenta, and yellow against deep purples and blacks
- **Style:** Isometric-inspired city blocks with glowing windows and holographic billboards
- **Tool:** Created with pixel-perfect precision for crisp edges at any scale
