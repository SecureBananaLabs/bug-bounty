# Pixel Art: Cyberpunk Cityscape

**What:** A 64x64 pixel art depiction of a rainy cyberpunk cityscape with neon-lit skyscrapers, holographic billboards, and reflective wet streets.

**Why:** I chose this theme to capture the contrast between organic rain and synthetic neon light, exploring how limited pixels can convey atmosphere, depth, and emotion through creative color palette choices and dithering techniques.

Created with creative thinking to evoke a futuristic, moody urban environment.
