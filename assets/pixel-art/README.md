# Cyberpunk Cityscape Pixel Art

**What:** A 64x64 pixel art depiction of a futuristic cyberpunk cityscape at night.

**Why:** I chose this theme because cyberpunk aesthetics blend vibrant neon colors with moody atmospheres, which makes for visually striking pixel art. The contrast between dark buildings and bright neon signs showcases the medium's ability to create depth and emotion with minimal pixels.

Created for the bug bounty pixel art challenge.
